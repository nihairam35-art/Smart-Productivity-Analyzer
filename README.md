# Smart Habit & Productivity Analyzer

##  Overview
This project analyzes daily habits such as study hours, sleep, screen time, and exercise to understand and predict productivity using Machine Learning.

##  Features
- Analyzes user habits and productivity patterns  
- Predicts productivity score using regression  
- Classifies users into productivity levels  
- Groups users using clustering (K-Means)  
- Provides simple recommendations to improve productivity  

##  Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

##  Machine Learning Techniques
- Linear Regression  
- Logistic Regression  
- Decision Tree  
- K-Means Clustering  

##  Key Insights
- Higher study hours and sleep improve productivity  
- High screen time negatively impacts performance  
- Users can be grouped into high, medium, and low productivity categories  

##  How to Run

2. Open in VS Code or Jupyter Notebook  
3. Run all cells step by step  

##  Conclusion
This project demonstrates an end-to-end machine learning workflow including data analysis, modeling, and generating actionable insights.
